# Plan 05-01: Migrate aer-search-aws-goes to pluggy hookimpl

## Objective

Rewrite the `search_aws_goes` plugin from the deprecated `@plugin(name=..., category="search")` + `SearchQuery` pattern to the new `@hookimpl` class pattern defined in `aer.plugin.core.AerSpec`.

### Key Changes

**Old signature:**
```python
@plugin(name="aws_goes", category="search")
def search_aws_goes(query: SearchQuery) -> GeoDataFrame["SearchResultSchema"]:
```

**New signature:**
```python
class AwsGoesSearchPlugin:
    @hookimpl
    def search(self, collections: list[str], intersects: GeomLike | None,
               time_range: TimeRange | None, search_params: dict | None = None) -> GeoDataFrame[SearchResultSchema]:
```

**Schema change:** The old `SearchResultSchema` had grid-exploded columns (row, col, epsg, channel, product_id, etc.). The new one has: `id`, `collection`, `geometry`, `start_time`, `end_time`, `href`.

**Removed concepts:**
- `Product` type → replaced by `collections: list[str]` (e.g., `["ABI-L1b-RadF"]`)
- `Channel` type → not part of search anymore; pass channel filters via `search_params`
- `SearchQuery` → replaced by individual args to hookspec
- `GridSpatialExtent` → replaced by `intersects: GeomLike | None`
- `@plugin` decorator → replaced by `@hookimpl`

<action>
Replace the entire `core.py` with a class-based plugin.

1. **Remove all deprecated imports:**
   - Remove: `from aer.plugin import plugin`
   - Remove: `from aer.search import SearchQuery, SearchResultSchema`
   - Remove: `from aer.spatial import GridSpatialExtent`
   - Remove: `from aer.spectral import Channel, Product`

2. **Add new imports:**
   ```python
   from aer.plugin import hookimpl, SearchResultSchema
   ```

3. **Remove deprecated helpers:**
   - Remove `_find_channel_by_id()` function
   - Remove `VALID_PRODUCTS` list and `_all_valid_products()` function
   - Keep `_parse_goes_filename()` — still needed for timestamp extraction

4. **Create class `AwsGoesSearchPlugin`** with a single method:
   ```python
   class AwsGoesSearchPlugin:
       @hookimpl
       def search(
           self,
           collections: list[str],
           intersects: GeomLike | None,
           time_range: TimeRange | None,
           search_params: dict | None = None,
       ) -> GeoDataFrame[SearchResultSchema]:
   ```

5. **Map parameters inside the method body:**
   - `collections` replaces `query.products` — iterate over collection name strings directly, e.g. `for collection in collections:`
   - `time_range` replaces `query.time_range` — same `TimeRange` type
   - Channel filtering: extract from `search_params.get("channels")` if present (list of channel ID strings like `["1", "13"]`)
   - Satellite filtering: extract from `search_params.get("satellites")` if present (list of satellite name strings like `["GOES-16"]`), otherwise default to all supported entries in `sat_to_bucket.keys()`

6. **Build result rows with new schema columns:**
   - `id` = unique row ID (the md5 hash currently computed as `unique_id`)
   - `collection` = collection name string (e.g., `"ABI-L1b-RadF"`)
   - `geometry` = granule footprint polygon. For GOES full-disk, use `intersects` geometry if provided, or a default GOES full-disk bounding box `Polygon([(-156, -81), (6, -81), (6, 81), (-156, 81)])` for GOES-East
   - `start_time` = from `_parse_goes_filename()`
   - `end_time` = from `_parse_goes_filename()`
   - `href` = S3 URL string (`f"s3://{f_path}"`)

   No more grid cell columns (row, col, epsg, dist, channel, name, etc.) — those are the extraction layer's responsibility now.

7. **Remove the grid cell explosion loop.** Instead of one row per (cell, channel), each S3 file becomes exactly one row.

8. **Optionally pass extra info via additional columns** (the schema has `strict=False`):
   - `https_url` = HTTPS URL (convenient for users)
   - `size_mb` = file size
   - `channel_id` = the band/channel ID from filename (e.g., `"1"`, `"13"`)

9. **Handle empty results:** Return empty GeoDataFrame with the required columns:
   ```python
   gdf = gpd.GeoDataFrame(columns=["id", "collection", "geometry", "start_time", "end_time", "href"], geometry="geometry")
   ```
</action>

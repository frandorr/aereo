# Quick Task 260339: Create OSCAR API consumer functions

## Goal
Create Python functions to consume the OSCAR API and convert satellite and instrument data into pandas DataFrames.

## must_haves
- Function to fetch all satellites from OSCAR API with pagination handling
- Function to fetch all instruments from OSCAR API with pagination handling
- DataFrame structure for satellites: id, slug, acronym, fullname, status, orbit, launch_date, EoL, space_agency, altitude, longitude, ECT, WIGOS_Station_Identifier, data_access_link
- DataFrame structure for instruments: id, slug, name, fullname, instrumenttype, providing_agency, satellite_id (FK), start_date, EoL, status, classification (flattened)
- Handle API pagination automatically
- Store functions in development/local/oscar.py

**Files:**

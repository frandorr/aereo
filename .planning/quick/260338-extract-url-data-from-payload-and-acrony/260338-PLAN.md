# Quick Task 260338: Extract URL data from Payload and Acronym column hyperlinks

## Goal
Create an extraction function to retrieve hyperlink URLs from Excel cells in the Payload and Acronym columns, and update the oscar.py marimo notebook to demonstrate its usage.

## must_haves
- Extraction function that reads `.hyperlink.target` from openpyxl cells
- Handles cases where hyperlinks may not exist (graceful fallback)
- Updated oscar.py notebook demonstrating the extraction
- Documentation of the approach

## Task 1: Create hyperlink extraction function

**Files:**
- `development/local/oscar.py` (update)

**Action:**
Create a function `extract_hyperlink_data(sheet, column_name)` that:
1. Finds the column by name in the sheet header
2. Iterates through all rows in that column
3. For each cell, checks if `cell.hyperlink` exists
4. If hyperlink exists, extracts `cell.hyperlink.target`
5. Returns a list of dicts with row data: `{'row': int, 'value': str, 'url': str or None}`

Add this function to the oscar.py notebook in a new cell.

**Verify:**
- Function accepts sheet and column_name parameters
- Returns properly structured data
- Handles cells without hyperlinks gracefully (url = None)
